# LlmSync

Простая консольная утилита, которая:

1. Запускает `LlmCall.exe`, передавая ему все свои аргументы командной строки, после чего дожидается его завершения (и сама завершается с тем же кодом возврата, что и `LlmCall.exe`).

2. Перенаправляет свои стандартные потоки в `LlmCall.exe`, позволяя ей сообщать об ошибках в консоль.
